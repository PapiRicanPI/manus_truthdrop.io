# Case Creation Process

## Add New Case Form

URL: **https://truthdrop.io/admin/add-case**

### Form Fields:

1. **Case Title*** (Required)
   - Example: "NYC Charity Fraud - $2.5M Embezzlement"
   - This is the main headline for the case

2. **Subtitle** (Optional)
   - Brief tagline or summary
   - Appears below the title

3. **Featured Image** (Optional)
   - File upload field
   - Max 5MB
   - Supported formats: JPG, PNG, GIF, or WebP recommended
   - This image appears as the case thumbnail

4. **Short Description*** (Required)
   - 1-2 sentence summary of the case
   - This appears in case listings and previews

5. **Full Case Details*** (Required)
   - Markdown supported
   - Should include:
     - Location
     - Amount stolen
     - Perpetrators
     - Fraud methods
     - Evidence
     - Legal outcomes
   - Formatting tips shown:
     - Use **Bold** for emphasis
     - Use ## Headers for sections
     - Use - Lists for bullet points

### Action Buttons:

- **Add Case** - Publish the case immediately
- **Cancel** - Return to admin dashboard without saving

### Markdown Support:

The Full Case Details field supports Markdown formatting:
- **Bold text**: `**text**`
- Headers: `## Header`
- Lists: `- Item`
