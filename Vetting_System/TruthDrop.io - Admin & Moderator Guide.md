## Introduction

This guide provides comprehensive instructions for administrators and moderators on how to manage and update the TruthDrop.io portal. It covers content management, user submissions, and administrative functions.

---

## User Roles

TruthDrop.io has three main user roles:

| Role | Permissions & Responsibilities |
|---|---|
| **Admin** | - Full control over the platform<br>- Create, edit, and delete all content<br>- Manage users and their roles<br>- Approve/reject vetting applications<br>- Review and publish tips<br>- Access all platform settings and analytics |
| **Moderator** | - Manage content and user submissions<br>- Edit and update case files<br>- Review and approve/reject tips<br>- Review vetting applications<br>- Cannot manage users or change platform settings |
| **User** | - Basic access to view published content<br>- Can submit tips and apply for vetting<br>- Cannot edit or manage any content |

## Admin Dashboard Overview

**URL:** https://truthdrop.io/admin

### Key Sections:

1. **Statistics** - At-a-glance view of total cases, users, published content, and admins.
2. **User Management** - View and manage user roles.
3. **Quick Actions** - Main navigation for all administrative tasks.

### Quick Actions Available:

- **Create New Case File** - Add a new case to the database
- **Manage Case Files** - Edit, update, or delete existing cases
- **Review Vetting Applications** - Approve/reject user access requests
- **Review Tips** - Review tips submitted by the public
- **Review Access Requests** - Manage access requests
- **Research Resources** - Access research materials
- **Review Draft Cases** - Review and publish AI-generated drafts
- **View Analytics** - View platform analytics

## Managing Case Files

### Creating New Case Files

**Who can do this?** Admins

1. **Navigate to Admin Dashboard**
2. **Click "Create New Case File"**
3. **Fill out the form:**
   - **Case Title*** (Required)
   - **Subtitle** (Optional)
   - **Featured Image** (Optional, max 5MB)
   - **Short Description*** (Required, 1-2 sentences)
   - **Full Case Details*** (Required, Markdown supported)
4. **Click "Add Case"** to publish immediately.

### Editing & Updating Cases

**Who can do this?** Admins, Moderators

1. **Navigate to Admin Dashboard**
2. **Click "Manage Case Files"**
3. **Select a case file** to open its detail page
4. **(Future Feature)** An "Edit" button will be available on the case detail page for admins/moderators to update content.

### Reviewing Draft Cases

**Who can do this?** Admins, Moderators

1. **Navigate to Admin Dashboard**
2. **Click "Review Draft Cases"**
3. **Review AI-generated drafts** from the news scanner
4. **Approve, reject, or edit** drafts before publishing.

## Managing User Submissions

### Reviewing Vetting Applications

**Who can do this?** Admins, Moderators

1. **Navigate to Admin Dashboard**
2. **Click "Review Vetting Applications"**
3. **Use the enhanced interface:**
   - **Review** applicant details
   - **Enter review notes** (required)
   - **Click "Approve" or "Reject"**
   - Emails are sent automatically

### Reviewing Tips

**Who can do this?** Admins, Moderators

1. **Navigate to Admin Dashboard**
2. **Click "Review Tips"**
3. **Click "Review"** on a pending tip
4. **In the review dialog:**
   - View tip details and evidence links
   - **Update status** (Under Review, Published, Rejected)
   - **Add review notes**
   - **Click "Update Status"** to save changes

## User Management

**Who can do this?** Admins

1. **Navigate to Admin Dashboard**
2. **In the User Management table:**
   - View all users and their roles
   - **Click the dropdown** next to a user's role
   - **Select a new role** (Admin, Moderator, User)
   - The change is saved automatically

## Best Practices

### For Admins:

- **Regularly review** pending tips and vetting applications.
- **Use strong passwords** and enable 2-factor authentication.
- **Be cautious** when assigning admin roles.
- **Periodically review** user roles and permissions.
- **Monitor analytics** to understand platform usage.

### For Moderators:

- **Follow content guidelines** when editing case files.
- **Provide clear review notes** for all decisions on tips.
- **Escalate complex cases** or suspicious tips to an admin.
- **Maintain a neutral and objective tone** in all content.
