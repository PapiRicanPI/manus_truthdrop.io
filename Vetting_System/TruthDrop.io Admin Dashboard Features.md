## Quick Actions Available

The admin dashboard provides the following quick action buttons:

1. **Create New Case File** - Add new poverty fraud case to the database
2. **Manage Case Files** - Edit, update, or delete existing cases
3. **Review Vetting Applications** - Approve/reject user access requests
4. **Review Tips** - Review submitted tips from the public
5. **Review Access Requests** - Manage access requests
6. **Research Resources** - Access research materials
7. **Review Draft Cases** - Review and publish draft case files
8. **View Analytics** - View platform analytics and statistics

## Role-Based Access

The system supports three user roles:
- **Admin** - Full access to all features
- **Moderator** - Can manage content and review submissions
- **User** - Basic access to view content
